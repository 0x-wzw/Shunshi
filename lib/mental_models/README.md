# Mental Models for BaZi Decision-Making

A comprehensive framework of cognitive tools derived from Chinese metaphysical principles, integrated into the BaZi Decision Intelligence system.

## Overview

This module transforms the Five Elements (五行) from ancient Chinese philosophy into practical mental models for modern decision-making. Each element represents not just an astrological concept, but a fundamental pattern of thinking and acting.

---

## Core Philosophy

> **"As above, so below. As in the cosmos, so in the mind."**

The Five Elements provide a unified framework:
- **Wood (木)** → Growth, Strategy, Vision
- **Fire (火)** → Passion, Expression, Transformation
- **Earth (土)** → Foundation, Timing, Nourishment
- **Metal (金)** → Precision, Division, Integrity
- **Water (水)** → Adaptation, Depth, Flow

## Decision Frameworks

### 1. Five Element Decision Matrix (五行决)
**When:** Complex decisions with multiple stakeholders or uncertain outcomes

**Process:**
1. **Wood:** What are the growth opportunities? What would I learn?
2. **Fire:** Where is the passion/energy? What would illuminate?
3. **Earth:** What is the timing? What foundation exists?
4. **Metal:** What must be eliminated? What are hard constraints?
5. **Water:** Where is the flow? What am I resisting?
6. **Synthesis:** Which element dominates? Align decision with that energy.

---

### 2. Seasonal Timing Framework (待时而动)
**When:** Deciding WHEN to act rather than WHETHER to act

**Seasonal Map:**
- **寅卯 Spring:** Plant seeds → Begin initiatives
- **巳午 Summer:** Expand → Maximize visibility
- **申酉 Autumn:** Harvest → Collect returns
- **亥子 Winter:** Store → Consolidate and plan

---

### 3. Ten God Interaction Model (十神应用)
**When:** Analyzing relationships, partnerships, or collaboration

**Relationship Patterns:**
- **印 (Support):** Who is producing me? ← Accept help
- **食伤 (Expression):** What am I producing? → Give value
- **财 (Wealth):** What am I overcoming? → Pursue opportunities
- **官杀 (Pressure):** What is overcoming me? → Respect constraints
- **比劫 (Peers):** Who is like me? ↔ Collaborate selectively

---

### 4. Branch Interaction Strategy (地支运筹)
**When:** Interpersonal dynamics or hidden factors complicate decisions

**Interaction Types:**
- **冲 Clash:** Fundamental opposition → Time separation
- **合 Combination:** Alliance energy → Leverage partnerships
- **害 Harm:** Hidden conflict → Address before eruption
- **刑 Punishment:** Repetitive patterns → Break cycles
- **会 Meeting:** Natural groupings → Align with these

---

### 5. Root and Branch Analysis (本末先后)
**When:** Overwhelmed by details; need leverage points

**Process:**
1. Identify "branches" — visible symptoms, immediate concerns
2. Trace to "roots" — underlying causes or sources
3. Classify: Is this a root issue or branch symptom?
4. **Prioritize:** Address roots before branches
5. **Resource allocation:** Invest most in root causes

## Cognitive Bias Detection

### Excessive Wood Bias (木亢)
**Pattern:** Growth without maintenance; constant expansion
**Bazi Correlation:** Strong Wood + Weak Earth
**Correction:** Force "maintenance seasons"; finish before starting

### Uncontrolled Fire Bias (火炎)
**Pattern:** Enthusiasm without follow-through; action addiction
**Bazi Correlation:** Strong Fire + Weak Water
**Correction:** Cooling-off periods; 10-year sustainability test

### Excessive Earth Bias (土滞)
**Pattern:** Analysis paralysis; over-preparation
**Bazi Correlation:** Strong Earth + Weak Wood
**Correction:** Decision deadlines; ship at 80%

### Excessive Metal Bias (金刚)
**Pattern:** Over-elimination; harsh boundaries
**Bazi Correlation:** Strong Metal + Weak Fire
**Correction:** 10% "inefficient" buffer; boundaries with warmth

### Scattered Water Bias (水流)
**Pattern:** Adaptability becomes non-commitment
**Bazi Correlation:** Strong Water + Weak Earth
**Correction:** Irrevocable commitments; define "you" beyond adaptation

### Missing Element Blindness (缺)
**Pattern:** Blind to weakest element qualities
**Bazi Correlate:** Element at <10%
**Correction:** Partner with strength in your weakness

## Implementation Notes

### Mental Model Structure:
```typescript
interface MentalModel {
    name: string;           // Display name + Chinese
    description: string;    // Core insight
    application: string[];  // Actionable steps
    triggers: string[];     // When to apply
    antidotes: string[];    // What to avoid
    origin: string;         // Classical source
}
```

### Decision Framework Structure:
```typescript
interface DecisionFramework {
    name: string;
    context: string;        // When to use
    steps: string[];        // Process
    elementAlignment: string[]; // Which elements
    contraindications: string[]; // When NOT to use
}
```
