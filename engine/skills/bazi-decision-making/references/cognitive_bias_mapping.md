# Five Elements to Cognitive Bias Mapping

This document maps Wuxing (五行) imbalances to identifiable cognitive biases in decision-making. 
Use this when analyzing decision charts to identify potential blindspots.

---

## Elemental Imbalance → Bias Framework

### EXCESSIVE WOOD (木过旺)

**Associated Bias: Growth Bias / Expansion Blindness**

**Symptoms:**
- Preference for growth over maintenance
- Adding features/projects before consolidating
- Viewing obstacles as temporary rather than structural
- Optimistic projections without risk buffers
- "We'll fix it in the next round" mentality

**Metaphysical Root:**
Wood represents growth, expansion, planning. Excessive wood creates
unchecked Yang expansion—like a tree growing without root stability.

**In Decision Context:**
```
Chart shows: Multiple wood stems/branches, spring season dominance
Risk: Over-committing resources to new initiatives
Blindspot: Existing system maintenance needs
```

**Corrective Strategies:**
1. Force "maintenance seasons" in planning cycles
2. Require maintenance budgets to equal growth budgets
3. Implement "root tests"—prove foundation before expansion
4. Counter-weight: Add Metal element activities (precise measurement, strict boundaries)

**Related Mathematical Bias:**
- Optimism bias
- Planning fallacy
- Scope creep acceptance

---

### DEFICIENT WOOD (木不足)

**Associated Bias: Stagnation Bias / Resistance to Growth**

**Symptoms:**
- Over-reliance on existing methods
- Dismissing new opportunities too quickly
- "We've always done it this way"
- Risk aversion disguised as prudence
- Maintenance without evolution

**Metaphysical Root:**
Wood deficiency means insufficient generative force—system loses
capacity to sprout new directions.

**Corrective Strategies:**
1. Schedule dedicated "exploration phases"
2. Set metrics for new initiative attempts (not just success)
3. Rotate roles to avoid calcification
4. Counter-weight: Add Wood element activities (strategic planning, vision work)

---

### EXCESSIVE FIRE (火过旺)

**Associated Bias: Action Bias / Visibility Priority**

**Symptoms:**
- Preference for visible action over quiet preparation
- Announcing before completing
- Meeting activity mistaken for progress
- Short burst energy without follow-through
- "Shipping" without adequate testing

**Metaphysical Root:**
Fire represents illumination, transformation, action. Excessive fire
creates unstable energy—like a fire that burns too bright, too fast.

**Corrective Strategies:**
1. Implement mandatory cooling-off periods
2. Value invisible preparation work publicly
3. Celebrate "not launching" when timing is wrong
4. Counter-weight: Add Water element activities (reflection, research, patience)

**Related Mathematical Bias:**
- Action bias (preference for action over inaction)
- Present bias (immediate gratification)
- Hot hand fallacy

---

### DEFICIENT FIRE (火不足)

**Associated Bias: Visibility Avoidance / Hidden Action**

**Symptoms:**
- Avoiding necessary communication
- Working in isolation too long
- Under-marketing good work
- Missing momentum moments
- Analysis without dissemination

**Metaphysical Root:**
Fire deficiency means insufficient illumination—good work remains obscured.

**Corrective Strategies:**
1. Build communication checkpoints into processes
2. Require public milestones
3. Assign "translators" to articulate vision
4. Counter-weight: Add Fire element activities (presentations, launches, celebrations)

---

### EXCESSIVE EARTH (土过旺)

**Associated Bias: Analysis Paralysis / Status Quo Entrenchment**

**Symptoms:**
- Endless deliberation before action
- Data collection as procrastination
- Committee design by inertia
- "We need more research" as eternal mantra
- Over-emphasis on harmony over clarity

**Metaphysical Root:**
Earth represents stability, support, digestion. Excessive earth creates
immobility—like soil too compacted for roots to penetrate.

**Corrective Strategies:**
1. Ship at 80% readiness
2. Set irreversible deadlines (not just targets)
3. Limit information gathering phases with hard stops
4. Counter-weight: Add Wood element activities (decision-making, new directions)

**Related Mathematical Bias:**
- Information bias
- Default effect (preferring current state)
- Choice overload

---

### DEFICIENT EARTH (土不足)

**Associated Bias: Foundation Neglect / Structural Blindness**

**Symptoms:**
- Building without adequate infrastructure
- Scaling before systems are ready
- Ignoring maintenance debt
- Assuming support systems will appear
- Undervaluing "boring" foundational work

**Metaphysical Root:**
Earth deficiency threatens structural integrity—buildings without foundations.

**Corrective Strategies:**
1. Mandate infrastructure audits before scaling
2. Allocate resources to maintenance proportional to growth
3. Measure system health, not just output
4. Counter-weight: Add Earth element activities (process documentation, system audits)

**Critical Decision Signal:**
If decision chart shows Earth as weakest element (especially 0 count),
yellow flag: infrastructure risk.

### EXCESSIVE METAL (金过旺)

**Associated Bias: Over-Reduction / Precision Obsession**

**Symptoms:**
- Cutting too deeply for efficiency
- Over-simplifying complex dynamics
- Binary thinking when spectrums exist
- Value destruction disguised as focus
- "Ruthless prioritization" without nuance

**Metaphysical Root:**
Metal represents precision, boundary-setting, cutting. Excessive metal
creates destructive reduction—like a blade that cuts healthy tissue.

**Corrective Strategies:**
1. Maintain 10% "inefficiency" buffer
2. Test cuts before permanent removal
3. Value retention alongside efficiency
4. Counter-weight: Add Water element activities (adaptation, flow states)

**Related Mathematical Bias:**
- Reductionist bias
- False precision
- Binary classification errors

### DEFICIENT METAL (金不足)

**Associated Bias: Precision Avoidance / Boundary Ambiguity**

**Symptoms:**
- Fuzzy success criteria
- Unclear commitments
- Difficulty saying "no"
- Scope ambiguity in projects
- Avoiding difficult conversations

**Metaphysical Root:**
Metal deficiency means insufficient clarity—boundaries dissolve, precision lost.

**Corrective Strategies:**
1. Define precise success/failure criteria upfront
2. Use kill criteria for projects
3. Practice clear boundary-setting
4. Counter-weight: Add Metal element activities (metrics definition, process refinement)

### EXCESSIVE WATER (水过旺)

**Associated Bias: Optionality Trap / Commitment Avoidance**

**Symptoms:**
- Keeping all doors open indefinitely
- Decisions delayed by "more options may appear"
- Portfolio bloat from hedging
- Flexibility worshipped over focus
- Analysis flowing without conclusion

**Metaphysical Root:**
Water represents flow, adaptability, wisdom. Excessive water creates
unbounded flow—commitment is avoided, options multiply indefinitely.

**Corrective Strategies:**
1. Make irrevocable commitments
2. Set option expiration dates
3. Value closure over flexibility
4. Counter-weight: Add Earth element activities (commitment, structure)

**Related Mathematical Bias:**
- Option value overestimation
- Regret aversion
- Maximizing vs satisficing imbalance

### DEFICIENT WATER (水不足)

**Associated Bias: Rigidity / Adaptation Failure**

**Symptoms:**
- Inability to pivot when conditions change
- Sticking to plan when plan fails
- Ignoring environmental signals
- Fixed mindset in dynamic contexts
- "Discipline" becoming dogma

**Metaphysical Root:**
Water deficiency means insufficient adaptability—the system rigidifies.

**Corrective Strategies:**
1. Build flexibility review points into plans
2. Practice intentional pivots
3. Value adaptation alongside consistency
4. Counter-weight: Add Water element activities (scenario planning, contingency prep)

## Detection Protocol

When analyzing a decision chart:

1. **Count elements** across 4 pillars (stems weighted 2x, branches 1x)
2. **Identify extremes:**
   - ≥4 counts = Excessive
   - 0 counts = Deficient (critical)
   - 1 count = Weak
3. **Map to biases** using this document
4. **Assess relevance** to decision type
5. **Recommend countermeasures**

### Example Analysis:

```
Chart: 甲丁己庚 / 辰卯酉午
Elements: Wood(3), Fire(3), Earth(2), Metal(2), Water(0)

Critical Finding: Water DEFICIENT (0 count)
Bias Risk: Rigidity / Adaptation Failure
Decision Context: Technology platform selection
Recommendation: 
  - Build flexibility review at 6 months
  - Create explicit pivot criteria
  - Avoid vendor lock-in commitments
```
